Integrating PostureAI Real-time Streams

Learn how to consume raw kinesiometric data through our low-latency WebSocket API. Designed for clinical monitoring and real-time biometric feedback loops.

The Biometric Payload

Every 100ms, the PostureAI engine broadcasts a unified health packet. This includes spinal curvature, head tilt, and shoulder symmetry indices.

{
  "session_id": "posture_7823_99",
  "biometrics": {
    "spinal_tilt": 12.4, // degrees
    "shoulder_alignment": 0.98, // symmetry score
    "tension_index": 42
  }
}

Data is sampled at 30Hz, smoothed using a Savitzky-Golay filter to remove physiological jitter without introducing phase lag.
